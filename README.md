# Tshering Sherpa — Personal Website

A dark, editorial personal website built with HTML, CSS, and vanilla JavaScript.  
Inspired by Kathmandu's night cityscape.

## 📁 Project Structure

```
tshering-sherpa-site/
├── index.html          # Main HTML
├── style.css           # All styles
├── script.js           # Cursor, scroll reveal, nav highlight
├── assets/
│   └── tshering.jpg    # Profile photo
└── README.md
```

## 🚀 Deploy to GitHub Pages

1. **Create a new GitHub repo** (e.g. `tshering-sherpa`)
2. **Upload all files** keeping the folder structure intact
3. Go to **Settings → Pages**
4. Under *Source*, select **Deploy from a branch → main → / (root)**
5. Click **Save** — your site will be live at:
   `https://<your-username>.github.io/tshering-sherpa/`

## ✏️ Customising

Open `index.html` and update:

| Section | What to change |
|---------|----------------|
| Hero | Title, subtitle, bio |
| About | Bio paragraphs, stat numbers |
| Skills | Skill names and descriptions |
| Experience | Roles, companies, dates, descriptions |
| Contact | Email address, LinkedIn URL |

Update `assets/tshering.jpg` to replace the profile photo — keep the same filename, or update the `src` in `index.html`.

## 🛠 Tech Stack

- Pure HTML5 / CSS3 / Vanilla JS — no frameworks, no build step
- Google Fonts: Cormorant Garamond + DM Sans
- CSS custom properties for theming
- IntersectionObserver for scroll reveals
- Custom animated cursor
